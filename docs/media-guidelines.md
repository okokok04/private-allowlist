# Media Evidence Guidelines

To support Level 3 submission evidence, capture the following:

1. Screenshot of the deployed app at the live demo URL:
   - https://mid-eta-ochre.vercel.app
   - Show the app UI after it loads successfully.
2. Short demo video showing:
   - wallet detection and connect flow
   - private proof generation
   - proof verification result shown in the UI

Suggested tools:
- Screenshot: browser native screenshot or a screen capture utility.
- Video: Loom, OBS, or any screen recorder that produces an MP4 link.

Once recorded, add the screenshot and video link to `README.md` or `SUBMISSION.md` as supporting documentation.
