# Demo Script

1. Open the app and explain the privacy problem.
2. Connect the wallet.
3. Generate a private membership proof.
4. Verify the proof.
5. Show that only the proof outcome is visible, not the identity.
6. Close by explaining the product potential for private communities and events.
